An introduction to DDD, CQRS and Event Sourcing

This talk gives an introduction to DDD, CQRS and Event Sourcing. It is aimed towards people that have no or limited experience with the topics. The goal is to get a first good understanding of the relevant concepts and patterns. The theoretical parts are complemented with various code examples.

Talk Outline:

  • Domains, Domain Models, Bounded Contexts
  • Software Architecture
  • Code quality
  • Value Objects, Entities, Services
  • Domain Events
  • Aggregates
  • Repositories
  • Application Services
  • CQRS
  • Event Sourcing
  • User Interface

If you'd like me to give the talk on a specific occasion, please contact me. The talk was held two times as a remote event. The overall duration is 3.5 hours (including breaks and Q&A).
